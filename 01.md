- Integrantes: Guilherme Harmatiuk Arantes

- Tecnologias Utilizadas: Wi-Fi. API.

- Objetivo:
    - Cada x segundos, o Arduino irá ler as informações dos sensores, e enviar ao backend através de uma API por Wi-Fi.
    - O Backend irá receber essas informações e gravar elas. 
    - Cada x sesugndos, o Frontend irá realizar uma requisição ao Backend, pedindo por essas informações.
    - Baseado nelas, o Frontend irá montar e atualizar um gráfico mostrando a corelação entre as variáveis e o tempo.

- Sensores Utilizados:
    - Gerantido, porque sei que tem na FAG:
        - DH11 (Umidade e Temperatura)
        - Sensor de luminosidade
    - Talvez (Não tem na FAG. Se conseguir comprar, pode usar eles, senão, vai só os de cima mesmo):
        - Sensor de intensidade UV (Photoresistor, mas pode ser que não funciona para medir uv)
        - Sensor de pressão atmosférica
